# ربات ارسال خودکار عکس در کانال تلگرام

## امکانات
- ارسال خودکار عکس در ساعت‌های مشخص
- انتخاب عکس‌ها به ترتیب یا تصادفی
- جلوگیری از تکرار عکس تا پایان مجموعه
- شروع دوباره مجموعه پس از تمام‌شدن عکس‌ها
- کپشن عمومی یا کپشن اختصاصی برای هر فایل
- منطقه زمانی ایران

## ۱) ساخت ربات
1. در تلگرام وارد `@BotFather` شوید.
2. دستور `/newbot` را بفرستید.
3. توکن دریافتی را فقط در فایل `.env` قرار دهید و برای دیگران نفرستید.
4. ربات را به کانال اضافه کنید.
5. ربات را ادمین کنید و دسترسی `Post Messages` بدهید.

## ۲) تنظیم فایل
1. فایل `.env.example` را کپی و نام نسخه جدید را `.env` بگذارید.
2. مقدار `BOT_TOKEN` و `CHANNEL_ID` را وارد کنید.
3. ساعت‌ها را در `POST_TIMES` تنظیم کنید.

برای کانال عمومی:
`CHANNEL_ID=@username_channel`

برای کانال خصوصی باید شناسه عددی کانال را پیدا کنید؛ معمولاً با `-100` شروع می‌شود.

## ۳) افزودن عکس
عکس‌ها را داخل پوشه `photos` بریزید.

اگر حالت ترتیبی باشد، نام‌گذاری پیشنهادی:
- `001.jpg`
- `002.jpg`
- `003.jpg`

## ۴) کپشن‌ها
در فایل `captions.txt`:
- اولین خط معمولی، کپشن عمومی است.
- کپشن اختصاصی هر عکس:
`001.jpg|متن مخصوص این عکس`

متغیرهای قابل استفاده:
- `{date}` تاریخ روز
- `{filename}` نام عکس

## ۵) نصب و اجرا روی ویندوز
در پوشه پروژه PowerShell یا CMD را باز کنید:

```bash
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python bot.py
```

تا وقتی پنجره باز و کامپیوتر روشن است، ربات کار می‌کند.

## ۶) تست فوری
برای تست، یکی از ساعت‌های `POST_TIMES` را روی چند دقیقه بعد تنظیم کنید، برنامه را اجرا کنید و منتظر همان ساعت بمانید.

## نکته مهم
برای فعالیت ۲۴ساعته، این پروژه باید روی سرور یا هاست همیشه‌روشن اجرا شود. روی لپ‌تاپ فقط زمانی کار می‌کند که دستگاه روشن و برنامه در حال اجرا باشد.
